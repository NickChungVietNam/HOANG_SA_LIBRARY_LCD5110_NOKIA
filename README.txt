Đây là thư viện điều khiển lcd 5110 nokia bằng arduino .
Thư viện hỗ trợ tất các hàm vẽ từ cơ bản : điểm, đường thẳng, hình tam giác, elip,.v.v hỗ trợ 2 Font chuẩn ASCII và Unicode dành cho tiếng Việt.
Thư viện hỗ trợ in ra số theo các kiểu số âm, số nguyên, số thực ..với 3 kiểu Font khác nhau , khả năng thay đổi cỡ chữ và số. Vẽ ảnh bitmap theo nhiều hiệu ứng khác nhau...
Phục vụ cho việc hiển thị đa chức năng với nhiều yêu cầu khác nhau.
